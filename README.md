# 🐱 CatBreak

A dancing green cat that forces you to take breaks.

## Install (Chrome/Edge)
1. Download this repo 
2. Open `chrome://extensions`
3. Turn on **Developer mode**
4. Click **Load unpacked** → select the `CatBreak` folder

## How it works
- At your set time, a dancing cat blocks the page
- You can't close it until break ends

# 🐱 CatBreak

一只跳舞的绿猫，强制你休息。

## 安装（Chrome/Edge）
1. 下载仓库
2. 打开 `chrome://extensions`
3. 打开 **开发者模式**
4. 点击 **加载已解压的扩展** → 选择 `CatBreak` 文件夹

## 玩法说明
- 设置好提醒间隔时间及休息倒计时
- 到时间后跳舞的猫挡住网页
- 休息结束前关不掉
